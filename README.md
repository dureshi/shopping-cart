# Shopping Cart Application

## 📌 Description
A React-based shopping cart application that allows users to browse products and manage their cart dynamically.

## 🚀 Features
- Add and remove items from cart
- Dynamic cart updates
- Component-based UI
- Responsive design

## 🛠️ Tech Stack
- React
- JavaScript
- CSS

## 📚 What I Learned
- React component structure
- State management using useState
- Handling user interactions
- Building scalable UI components
